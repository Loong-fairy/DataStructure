8.递归

一、什么是递归?

递归是一种解决问题的方法，将问题分解为更小的子问题，直到得到-一个足够小的问题可以被很简单的解决。

通常递归涉及函数调用自身。

二、整数列表的和

def listSum(numList):  # 递归函数
    if len(numList) == 1:
        return numList[0]
    return numList[0] + listSum(numList[1:])

三、递归的三大定律

  1. 递归算法必须具有基本情况:基本情况是算法停止递归的条件。

    通常是足够小，可以直接求解的问题。

    在整数列表的和中，基本情况是长度为1的列表

  2. 递归算法必须改变其状态并向基本情况靠近

  3. 递归算法必须以递归方式调用自身
